# OSM2SortedGeoJSON

## Requirements

This project uses CMake (https://cmake.org/) and Conan (https://conan.io/index.html).
All dependencies are automatically resolved by Conan except libosmium (https://osmcode.org/libosmium/)

## How to compile

	make
